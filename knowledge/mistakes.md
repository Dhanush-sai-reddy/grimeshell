# Mistake Log

Agents log failures here to prevent repeating them.
Before performing risky operations, agents should scan this file for relevant patterns.

## Template

```
### [DATE] — [SHORT DESCRIPTION]
- **Error**: What happened
- **Root Cause**: Why it happened
- **Fix**: What was done
- **Prevention**: How to avoid in future
- **Severity**: Low | Medium | High | Critical
- **Tags**: [comma-separated keywords for searchability]
```
